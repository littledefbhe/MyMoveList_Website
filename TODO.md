# Testing Checklist for Movie Title Font Size and UI Consistency

1. **Movie Title Font Size and Appearance**
   - Verify the movie title font size is 8px on all pages where `.movie-title` is used.
   - Ensure text is readable and does not overlap or get cut off.
   - Check hover state color changes correctly.

2. **Movie Poster and Card Layout**
   - Confirm movie poster images display correctly and maintain aspect ratio.
   - Validate movie card containers maintain consistent size and spacing.
   - Test responsiveness of cards on various screen sizes.

3. **Button Functionality and UI**
   - Test watchlist and rate buttons for visual feedback on hover and click.
   - Confirm button layout is consistent and accessible.

4. **Responsive Design**
   - Check font sizes, card layout, and button spacing on mobile, tablet, and desktop widths.
   - Validate media query overrides function as expected without unintended cascade.

5. **Navigation and Overall UX**
   - Test movie card links navigate correctly.
   - Verify no layout shifts or UI glitches during interaction.
